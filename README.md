# Pixel Dungeon Phaser TypeScript V1

Erste Migration von Pixel Dungeon auf Phaser 3 + TypeScript + WebGL.

## Lokal starten

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

Für GitHub Pages später den `dist/`-Ordner deployen.

## Enthalten

- Phaser 3 + TypeScript + WebGL
- Menü
- Map aus deinen alten Leveldaten
- Spielerbewegung
- einfache Kollisionen
- einfache Monsterbewegung
- deine Audio-Dateien unverändert in `public/assets/audio/`
- Mobile Touch Controls
- AudioManager mit Loops und Steps

## Noch nicht komplett portiert

- altes Lichtsystem
- alle Levelwechsel/Schlüssel-Tür-Details
- finaler Monster-AI-Feinschliff
- Pixel-Sprite-Animationen
- voll polierte UI
